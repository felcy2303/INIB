# MERN To-Do List Application

## Project Description

A Full Stack To-Do List application built using the MERN Stack.

## Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB
- Bootstrap
- Axios

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Responsive User Interface
- MongoDB Database Integration

## Folder Structure

## Installation

### Backend

```bash
cd server
npm install
node server.js
```

### Frontend

```bash
cd client
npm install
npm run dev
```
